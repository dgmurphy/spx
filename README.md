# spx
Express React spaCy

## Setup
Install python3

Create venv and install spaCy

Download spacy models

e.g. python -m spacy download en_core_web_sm (or the larger one for ner)


## To launch express:   
cd spx-api

npm install

npm run debug

in vscode -> attach to node process

## To launch react
cd spx-client

npm install

npm start
